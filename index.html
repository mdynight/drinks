Cocktail Book

Project Overview

Cocktail Book is a mutli-user Content Management System containing an end-user front-end for viewing database items, and a author-user back-end for entering database items.

Page One
/index.php

Renders a list of cocktail recipes accessed from a database. It has search capability and a meaurement toggle button.

Page Two
/login.php

Web form for entering records to a database, if a user is registered.

      Page ii. 'User'

      The page only display a log-in option if the user is not logged-in already.

      Log in to add drinks to database


      Once the user is logged in, a Web form is displayed. HTML, serverside, and Javascript validation are required for best user-experience. Once the fields are filled and meet requirements the user can click the submit button to run the script which will POST the values to the database. Some fields have datasets as value options. There is an and/or boolean field which is activated by buttons, and reveals a new input on click. The button following the ingredient input will reveak a new input on click.







      Once submitted confirmation will be displayed on a new page. Ooptions to return to index.php or user.php will be shown

      Thanks, the recipe was submiited successfully.
